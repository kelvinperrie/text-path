How to use

  1. Enter some text here that you want to move along a path
  2. If you want, change the speed increment (I'm not your dad, but probably don't)
  3. Click the button
  4. Move your mouse to where you want the text to start from and click the mouse button to start recording a path
  5. Move the mouse in the path you want
  6. Click the mouse button to stop recording the path. The path will be outputted here
  7. Click the to see the text more along the path

How to put on a website (????)

  2. Add jquery, text-path.js, and text-path.css to your website
